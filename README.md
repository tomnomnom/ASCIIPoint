# ASCIIPoint

An ASCII presentation tool by Gareth Halfacree and Tom Hudson. 
Written in less than 24 hours for LeedsHack 2012.

## Requirements 
* linux
* PHP 5.4
* jp2a

## Instructions
Run `./present.sh` to see the example slides.
